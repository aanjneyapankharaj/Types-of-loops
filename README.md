# Types-of-loops

In this experiment, we studied and implemented different types of loops in Python to perform repetitive tasks efficiently. The experiment mainly focused on the while loop and for loop, along with control statements like break and continue.

First, we learned the while loop, which executes a block of code repeatedly as long as a specified condition is true. Using the while loop, we performed several tasks such as printing numbers from 1 to 5, printing numbers from 1 to N, calculating the factorial of a number, generating the Fibonacci series, reversing a number, checking whether a number is a palindrome, and counting the number of digits in a given number.

Next, we studied loop control statements. The break statement was used to terminate the loop when a certain condition is met, even if the loop condition is still true. The continue statement was used to skip the current iteration and move to the next iteration of the loop. Using these statements, we created programs such as printing numbers from 1 to 10 while skipping 5 and printing only odd numbers.

After that, we explored the for loop, which is commonly used to iterate over sequences such as ranges, lists, or strings. Using for loops, we printed even numbers from 1 to 10, calculated the sum of the first N natural numbers, and implemented nested loops to display combinations of numbers.

We also used nested loops for working with matrices, including displaying a 3×3 matrix and performing matrix multiplication. Additionally, we created a program that accepts three digits and prints all possible combinations.

Further, we implemented number-based programs such as checking whether a number is an Armstrong number and printing all prime numbers within a given range.

Finally, we used loops to generate different pattern printing programs, including inverted triangle, triangle, right-angled triangle, hourglass pattern, and Pascal’s triangle. These programs helped us understand the use of nested loops and spacing logic for creating patterns.

Overall, this experiment helped us understand how loops work in Python and how they can be applied to solve mathematical problems, work with matrices, generate number sequences, and create pattern designs efficiently.
